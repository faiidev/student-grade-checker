# 🎓 Student Grade Checker V1

## 📖 Description

Student Grade Checker V1 adalah mini project Python yang dibuat untuk mempelajari konsep percabangan menggunakan `if`, `elif`, dan `else`. Program ini menerima input berupa nama dan nilai mahasiswa, kemudian menentukan grade serta status kelulusan berdasarkan nilai yang dimasukkan.

---

## ✨ Features

- Input nama mahasiswa
- Input nilai mahasiswa
- Menentukan grade (A, B, C, D, E)
- Menentukan status kelulusan (LULUS / TIDAK LULUS)
- Menampilkan hasil dengan format yang rapi

---

## 📚 Materi yang Dipelajari

- Variable
- Input & Output
- Operator Perbandingan
- Percabangan (`if`, `elif`, `else`)
- Boolean (`True` dan `False`)

---

## 💻 Contoh Output

```text
========================
STUDENT GRADE CHECKER
========================

Masukkan Nama  : Faii
Masukkan Nilai : 88

========================
Nama     : Faii
Nilai    : 88
Grade    : B
Status   : LULUS
========================
```
